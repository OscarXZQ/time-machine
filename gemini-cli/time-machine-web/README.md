# ChronoVisor - Time Machine Simulator

A visually stunning, browser-based time travel simulator built with Next.js, React Three Fiber, and Tailwind CSS.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

## Installation

1.  Navigate to the project directory:
    ```bash
    cd time-machine-web
    ```

2.  Install the dependencies:
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

## Launching the Simulator

1.  Start the development server:
    ```bash
    npm run dev
    # or
    yarn dev
    ```

2.  Open your browser and navigate to:
    `http://localhost:3000`

## Controls

-   **Target Year:** Use the controls near the central year display to adjust your destination.
-   **Initiate Jump:** Click the "INITIATE JUMP" button to trigger the warp sequence.
-   **Visuals:** Enjoy the 3D particle warp tunnel and cinematic interface effects.

## Features

-   **Immersive 3D Background:** Powered by Three.js and React Three Fiber.
-   **Cinematic UI:** Glassmorphism, neon accents, and smooth Framer Motion animations.
-   **Interactive State:** Real-time system logs, cooldown sequences, and dynamic starfields.

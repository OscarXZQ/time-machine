# Time Machine Simulator

A visually stunning, immersive time-machine simulator with rich animations, cinematic transitions, and futuristic effects.

## Features

- **Interactive Time Controls**: Adjust year, month, and day with smooth sliders
- **Real-time Display**: Current and target time displays with live updates
- **Cinematic Portal**: Animated portal with rotating rings and particle effects
- **Time Travel Animation**: Smooth transitions with visual effects when traveling through time
- **Futuristic UI**: Holographic overlays, glowing effects, and sci-fi aesthetics
- **Particle Effects**: Dynamic particle explosions and ambient effects
- **3D Interactions**: Mouse-responsive 3D tilt effects
- **Timeline Visualization**: Visual representation of temporal position
- **Data Streams**: Animated holographic data streams in the background

## Getting Started

1. Open `index.html` in a modern web browser
2. Use the sliders to select your target date
3. Click "INITIATE TIME TRAVEL" to begin the journey
4. Watch the portal activate and particles explode as you travel through time

## Controls

- **Year Slider**: Select any year from 1900 to 2100
- **Month Slider**: Choose month (1-12)
- **Day Slider**: Select day (automatically adjusts for month length)
- **INITIATE TIME TRAVEL**: Start the time travel sequence
- **RESET**: Return to current time and reset all controls

## Technical Details

- Pure HTML, CSS, and JavaScript (no dependencies)
- CSS animations and transitions for smooth effects
- Canvas-free particle system using DOM elements
- Responsive design for various screen sizes
- Modern browser features: CSS Grid, Flexbox, CSS animations

## Browser Compatibility

Works best in modern browsers that support:
- CSS animations and transitions
- CSS transforms and 3D transforms
- ES6 JavaScript features

## Customization

Feel free to customize:
- Colors in `:root` CSS variables
- Animation speeds and durations
- Particle counts and effects
- Portal size and appearance
